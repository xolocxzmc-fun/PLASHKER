# Сборка Windows-установщика через GitHub (с Mac, бесплатно)

Идея простая: ты загружаешь свой код на GitHub, а бесплатный робот GitHub
собирает установщик на **настоящей Windows** в облаке и даёт тебе скачать
готовый `Plashker-Setup.exe`. Никакой Windows на твоём Mac не нужно.
Сборка получается **x64** — запустится у всех обычных пользователей.

Я уже всё подготовил в проекте:
- `.github/workflows/build-windows.yml` — инструкция для робота (собирает сам)
- `Plashker-windows.spec`, `Plashker-Installer.iss` — конфиги сборки
- `AppIcon.ico`, `DocIcon.ico` — иконки
- `.gitignore` — настроен, чтобы не загружать лишнее

Тебе осталось сделать 3 вещи: создать репозиторий, загрузить код, скачать результат.

---

## Шаг 0. Проверь, что установлен Git (один раз)

Открой **Terminal** на Mac (Spotlight → напиши `Terminal` → Enter) и введи:
```bash
git --version
```
Если покажет версию — отлично. Если Mac предложит «установить инструменты
командной строки» — согласись и дождись установки, потом продолжай.

---

## Шаг 1. Создай аккаунт и пустой репозиторий на GitHub

1. Зарегистрируйся (если ещё нет) на https://github.com — бесплатно.
2. Нажми справа вверху **«+» → New repository**.
3. Заполни:
   - **Repository name:** `plashker` (или любое имя)
   - **Public** — оставь публичным. Для публичных репозиториев сборка
     бесплатна без лимитов. (Приватные тоже дают бесплатные минуты, но с публичным проще.)
   - **НЕ** ставь галочки «Add README / .gitignore / license» — репозиторий
     должен быть пустой.
4. Нажми **Create repository**.
5. На открывшейся странице скопируй адрес репозитория — он вида
   `https://github.com/ТВОЙ_ЛОГИН/plashker.git`

---

## Шаг 2. Загрузи код (команды в Terminal на Mac)

Зайди в папку проекта (это та, где лежит `run.py`):
```bash
cd "/Users/igor_morozov/Documents/FUN/PLASH_KER"
```

Инициализируй git и сделай первый «слепок» кода:
```bash
git init
git add .
git commit -m "Plashker: проект + автосборка Windows-установщика"
git branch -M main
```

Привяжи свой репозиторий на GitHub (подставь СВОЙ адрес из шага 1) и загрузи:
```bash
git remote add origin https://github.com/ТВОЙ_ЛОГИН/plashker.git
git push -u origin main
```
> При первом push GitHub попросит войти — откроется окно браузера, подтверди вход.
> Если попросит логин/пароль в терминале: логин — твой ник, вместо пароля нужен
> **Personal Access Token** (Settings → Developer settings → Tokens). Но обычно
> проще войти через всплывающее окно браузера.

---

## Шаг 3. Дождись сборки и скачай установщик

1. Открой свой репозиторий на GitHub в браузере.
2. Вверху нажми вкладку **Actions**.
3. Увидишь запущенную сборку **«Build Windows Installer»** с крутящимся
   жёлтым кружком. Идёт ~3–7 минут. Зелёная галочка = готово.
4. Кликни по этой сборке. Внизу страницы, в разделе **Artifacts**, будет
   файл **`Plashker-Setup`** — нажми на него, скачается zip.
5. Распакуй zip → внутри лежит **`Plashker-Setup.exe`**.

**Это твой готовый установщик.** Отдаёшь его любому человеку с Windows 10 или 11 —
он запускает, жмёт «Далее, Далее, Установить». Готово.

---

## Как пересобрать после изменений в коде

Просто снова загрузи изменения — робот пересоберёт всё сам:
```bash
cd "/Users/igor_morozov/Documents/FUN/PLASH_KER"
git add .
git commit -m "что поменял"
git push
```
Через несколько минут во вкладке **Actions** будет новый `Plashker-Setup.exe`.

Можно и без изменений: вкладка **Actions → Build Windows Installer →
кнопка «Run workflow»** запустит сборку вручную.

---

## Если что-то пошло не так

**Сборка во вкладке Actions «красная» (упала)** — кликни по ней, найди
красный шаг, раскрой его и посмотри последние строки ошибки. Чаще всего это
опечатка в зависимостях. Пришли мне текст ошибки — разберём.

**Приложение после установки открывается пустым белым окном** — на той Windows
не хватает **WebView2 Runtime**. На Win 11 он есть всегда, на Win 10 ставится с
https://developer.microsoft.com/microsoft-edge/webview2/ (Evergreen Standalone x64).

**SmartScreen/антивирус ругается при запуске установщика** — это нормально для
неподписанного `.exe`. «Подробнее → Выполнить в любом случае». Убрать совсем —
только платной подписью кода, для раздачи друзьям не обязательно.
